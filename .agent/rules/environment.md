## 注意事项

1. **远程开发**: 代码直接在云服务器上编辑和运行

3. **协作功能**: 当前禁用 (`collaboration.enabled: false`)，启用会导致编辑器渲染循环问题
4. **无 .env 文件**: API 密钥等需要单独配置
